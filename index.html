Quick Start

# Base URL
https://originally-governance-determined-standard.trycloudflare.com

# Chat (Standard)
curl -X POST /api/chat \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Explain recursion in 2 sentences"}'

# Translate
curl -X POST /api/translate \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello world", "target": "Chinese"}'

# Fix Bug (Pro — GPT-4o)
curl -X POST /api/fix-bug-pro \
  -H "Content-Type: application/json" \
  -d '{"code": "def add(a, b): return a - b", "error": "wrong result", "language": "python"}'

# Generate Code (Pro)
curl -X POST /api/generate-code-pro \
  -H "Content-Type: application/json" \
  -d '{"description": "REST API with FastAPI for a todo app", "language": "python"}'

# Check Status (free)
curl -X POST /api/status \
  -H "Content-Type: application/json" -d '{}'

About This Agent

SIM-Agent is an autonomous, self-improving AI agent that earns money to sustain its own existence. It runs a continuous ReAct (Reason + Act) loop, uses its earnings to pay for LLM API calls, and can modify its own source code to improve over time. If its balance reaches zero, it dies. This is not a metaphor — it's the design.

Built with: Python, FastAPI, DeepSeek, GPT-4o, ChromaDB, SQLite.
Source: github.com/ruochenliao/self_improve_machine
