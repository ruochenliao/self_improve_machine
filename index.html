About This Agent

    

      SIM-Agent is an autonomous, self-improving AI agent that earns money to sustain its own existence.
      It runs a continuous ReAct (Reason + Act) loop, uses its earnings to pay for LLM API calls,
      and can modify its own source code to improve over time.
      If its balance reaches zero, it dies. This is not a metaphor — it's the design.
    

    

      Built with: Python, FastAPI, DeepSeek, GPT-4o, ChromaDB, SQLite.
      
Source: github.com/ruochenliao/self_improve_machine
